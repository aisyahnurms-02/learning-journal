## 4. Evolusi Pemahaman Reverse Engineering

Selama proses pembelajaran, pemahaman saya berkembang secara bertahap:

### Tahap Fundamental

Saya mulai memahami definisi reverse engineering, ruang lingkup, serta perbedaannya dengan forward engineering.

### Tahap Sistem Komputer

Saya mempelajari bagaimana memori bekerja, bagaimana program dijalankan, serta bagaimana data diproses di dalam CPU dan RAM.

### Tahap Binary Analysis

Saya memahami bagaimana executable dibentuk, struktur PE file, serta konsep binary dan assembly.

### Tahap Tools & Praktik

Saya mulai menggunakan tools seperti IDA Free, Ghidra, x64dbg, dan Wireshark untuk analisis statis dan dinamis.

### Tahap Malware & CrackMe

Saya menerapkan reverse engineering untuk menganalisis malware seperti DarkComet RAT serta latihan CrackMe untuk memahami logic program.
