# 🧠 Learning Journal — Reverse Engineering

## 📌 Deskripsi Repository

Repository ini berisi dokumentasi proses pembelajaran saya selama mengikuti mata kuliah **Reverse Engineering** selama satu semester.

Tujuan utama dari repository ini adalah untuk mencatat perkembangan pemahaman terhadap konsep, tools, dan teknik dalam reverse engineering, mulai dari level dasar hingga analisis binary dan malware secara praktis.

Setiap folder mewakili catatan pembelajaran mingguan yang berisi:
- Materi yang dipelajari
- Ringkasan pemahaman
- Insight pribadi
- Hal yang masih ingin dipelajari
- Refleksi pembelajaran

---

## 🎯 Topik yang Dipelajari

## 🧩 Ringkasan Pembelajaran

### Week 01 — Introduction & Ethics
Pengenalan reverse engineering, ruang lingkup, sejarah, serta aspek etika dan legalitas.

### Week 02 — Memory Architecture
Pemahaman cara kerja memori (CPU, RAM, stack, heap) dalam eksekusi program.

### Week 03 — Binary & Executable
Struktur executable, format file (PE, ELF), serta konsep endianness.

### Week 04 — Reverse Engineering Tools
Pengenalan tools seperti IDA, Ghidra, x64dbg, Wireshark, dan lainnya.

### Week 05 — Malware Analysis Workflow
Tahapan malware analysis: triage, static analysis, dynamic analysis, hingga IOC.

### Week 06 — Malware Case Study (DarkComet RAT)
Analisis malware jenis RAT dan bagaimana sistem dapat dikendalikan dari jarak jauh.

### Week 07 — CrackMe Analysis
Latihan memahami logic autentikasi dan conditional jump dalam binary.

### Week 08 — Final Documentation
Penyusunan seluruh pembelajaran menjadi learning journal terstruktur.
